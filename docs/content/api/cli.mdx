# CLI 工具

终端输出美化与辅助函数。

## 依赖说明

本模块所有函数均依赖 `click`（bykpy 运行时自带），无需额外安装。

:::warning 唯一例外
`copy_to_clipboard` 需要 `pyperclip`。使用该函数的插件请在 `pyproject.toml` 中添加：

```toml
dependencies = [
    "pyperclip>=1.9.0",
]
```

缺失时 bykpy 会抛出清晰的 `ImportError` 并提示安装命令。
:::

## check_port

检查端口是否可用，不可用时打印友好的错误信息。

<Tabs>
<Tab label="函数签名">
```py
def check_port(
    port: int,
    host: str = "0.0.0.0",
    output_prefix: str = " ",
    silent: bool = False,
) -> bool:
    """
    Args:
        port: 要检查的端口号。
        host: 绑定地址，默认 0.0.0.0。
        output_prefix: 错误输出前缀，用于对齐。
        silent: True 时仅返回布尔值，不打印任何内容。

    Returns:
        True 表示端口可用。
    """
```
</Tab>
<Tab label="使用示例">
```py
from bykpy.api import check_port

# 基础用法 — 端口不可用时自动打印错误
if not check_port(8080):
    return

# 静默检查 — 仅返回布尔值
if check_port(3000, silent=True):
    print("端口可用")
```
</Tab>
</Tabs>

## colored_key_value

生成带 ANSI 颜色的「键: 值」字符串。

<Tabs>
<Tab label="函数签名">
```py
def colored_key_value(
    key: str,
    value: Any,
    key_color: str | None = "cyan",
    value_color: str | None = "yellow",
) -> str:
    """
    Args:
        key: 键文本。
        value: 值（自动转为字符串）。
        key_color: 键的颜色名称，None 表示无颜色。
        value_color: 值的颜色名称，None 表示无颜色。

    Returns:
        已着色（含 ANSI 转义序列）的字符串，可直接传给 click.echo()。
    """
```
</Tab>
<Tab label="使用示例">
```py
import click
from bykpy.api import colored_key_value

click.echo(colored_key_value("Status", "running", key_color="green"))
click.echo(colored_key_value("Local", "http://localhost:8080", key_color=None, value_color="cyan"))
```
</Tab>
</Tabs>

## echo_network_urls

打印本地和局域网可访问的 URL。

<Tabs>
<Tab label="函数签名">
```py
def echo_network_urls(
    networks: list[dict[str, Any]],
    port: int,
    include_virtual: bool = False,
) -> None:
    """
    Args:
        networks: get_private_networks() 的返回值。
        port: 服务端口号。
        include_virtual: 是否输出虚拟网卡（docker/vmware 等）的地址。
    """
```
</Tab>
<Tab label="使用示例">
```py
from bykpy.api import echo_network_urls, get_private_networks

networks = get_private_networks()
echo_network_urls(networks, port=8080)
# 输出：
#  Local: http://localhost:8080
#  Local: http://127.0.0.1:8080
#  [en0] Network URL:: http://192.168.1.100:8080

# 包含虚拟网卡
echo_network_urls(networks, port=8080, include_virtual=True)
```
</Tab>

## copy_to_clipboard

将文本复制到系统剪贴板。依赖 `pyperclip`，缺失时抛 `ImportError`。

<Tabs>
<Tab label="函数签名">
```py
def copy_to_clipboard(
    text: str,
    label: str = "URL",
    output_prefix: str = " ",
    silent: bool = False,
) -> None:
    """
    Args:
        text: 要复制的文本。
        label: 成功/失败提示中的标签名。
        output_prefix: 输出行前缀，用于对齐。
        silent: True 时静默执行，不输出任何提示。
    """
```
</Tab>
<Tab label="使用示例">
```py
from bykpy.api import copy_to_clipboard

# 复制 URL 并打印确认
copy_to_clipboard("http://192.168.1.100:8080")
# 输出:  URL has been copied to clipboard

# 静默复制
copy_to_clipboard("some text", silent=True)

# 自定义标签
copy_to_clipboard("token=abc123", label="Token")
# 输出:  Token has been copied to clipboard
```
</Tab>
</Tabs>

## wait_for_server_ready

轮询直到 HTTP 服务就绪（返回 200）。

:::tip 纯标准库
无额外依赖，始终可用。
:::

<Tabs>
<Tab label="函数签名">
```py
def wait_for_server_ready(
    port: int,
    host: str = "127.0.0.1",
    timeout: float = 10.0,
) -> bool:
    """
    Args:
        port: 服务端口。
        host: 服务地址，默认 127.0.0.1。
        timeout: 最长等待秒数。

    Returns:
        True 表示在超时前收到了 200 响应。
    """
```
</Tab>
<Tab label="使用示例">
```py
import threading
import webbrowser
from bykpy.api import wait_for_server_ready

# 典型场景：后台启动服务器，就绪后打开浏览器
def _auto_open(port):
    if wait_for_server_ready(port):
        webbrowser.open(f"http://localhost:{port}")

threading.Thread(target=_auto_open, args=(8080,), daemon=True).start()
# start_server(port=8080)  # 阻塞调用
```
</Tab>
</Tabs>
